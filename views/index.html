Timestamp Microservice

Example Usage:

/api/2015-12-25
/api/1451001600000
/api (returns current time)

Example Output:

{"unix":1451001600000,"utc":"Fri, 25 Dec 2015 00:00:00 GMT"}

API Endpoints:

  • GET /api/:date? - Returns timestamp information
    • If :date is empty, returns current time
    • If :date is a valid date string, returns the timestamp
    • If :date is a Unix timestamp (milliseconds), returns the date
    • If :date is invalid, returns {"error":"Invalid Date"}

Valid Date Formats:

  • Unix timestamp in milliseconds: 1451001600000
  • Date strings: 2015-12-25, December 25, 2015, 25 Dec 2015

Response Format:

Success:
{"unix": [timestamp in milliseconds], "utc": "[UTC date string]"}

Error:
{"error": "Invalid Date"}
